# Local Development

## Prerequisites

- Node.js 20+
- Docker Desktop or Docker Engine
- npm 10+

## Setup

```bash
cp .env.example .env
npm install
npm run docker:up
npm run seed
```

Start all app processes (auto-restarts on file changes via nodemon + package watchers):

```bash
npm run dev:apps
```

Or start infra and apps together:

```bash
npm run dev
```

## Enqueue a sample job

```bash
curl -X POST http://localhost:3001/api/jobs/enqueue-sample \
  -H 'Content-Type: application/json' \
  -H 'x-session-token: <token-from-login>' \
  -d '{"note":"hello"}'
```

Login first:

```bash
curl -X POST http://localhost:3001/api/auth/login \
  -H 'Content-Type: application/json' \
  -d '{"email":"admin@example.com","password":"change-me"}'
```

Use the password from `SEED_ADMIN_PASSWORD` in `.env`.
