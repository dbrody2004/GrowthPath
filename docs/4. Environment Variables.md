# Environment Variables

Copy `.env.example` to `.env` for local development.

## Core

| Variable | Required | Description |
| --- | --- | --- |
| `APP_NAME` | No | Application identifier (`growthpath`) |
| `NODE_ENV` | No | `development`, `test`, or `production` |
| `LOG_LEVEL` | No | Pino log level |

## API

| Variable | Required | Description |
| --- | --- | --- |
| `API_HOST` | No | Bind host (default `0.0.0.0`) |
| `API_PORT` | No | API port (default `3001`) |
| `CORS_ORIGIN` | No | Allowed browser origin |
| `SESSION_SECRET` | Yes | Session token hashing secret (16+ chars) |

## MongoDB

| Variable | Required | Local default | Production |
| --- | --- | --- | --- |
| `MONGODB_URI` | Yes | `mongodb://127.0.0.1:27118/growthpath` | `mongodb://mongo:27017/growthpath` |

## RabbitMQ

| Variable | Required | Local default | Production |
| --- | --- | --- | --- |
| `RABBITMQ_URL` | Yes | `amqp://127.0.0.1:5674` | `amqp://rabbitmq:5672` |
| `RABBITMQ_PREFETCH` | No | `10` | `10` |

## MinIO / S3

| Variable | Required | Description |
| --- | --- | --- |
| `S3_ENDPOINT` | Yes | S3-compatible endpoint URL |
| `S3_REGION` | No | AWS region string |
| `S3_ACCESS_KEY_ID` | Yes | Access key |
| `S3_SECRET_ACCESS_KEY` | Yes | Secret key |
| `S3_BUCKET` | Yes | Bucket name |
| `S3_FORCE_PATH_STYLE` | No | Use path-style URLs for MinIO |

## Seed admin

| Variable | Required | Description |
| --- | --- | --- |
| `SEED_ADMIN_EMAIL` | No | Admin email for `npm run seed` |
| `SEED_ADMIN_PASSWORD` | No | Admin password for local seed |

## UI / TLS

| Variable | Required | Description |
| --- | --- | --- |
| `VITE_API_URL` | No | API base URL for Vite dev (empty uses proxy) |
| `DEFAULT_DOMAIN` | Prod | Public domain for Nginx/Certbot |
| `CERTBOT_EMAIL` | Prod | Let's Encrypt contact email |
| `ENABLE_CERTBOT` | No | Set `true` to run Certbot in UI container |

## Docker build / deploy

| Variable | Default | Description |
| --- | --- | --- |
| `REGISTRY` | `ghcr.io` | Container registry |
| `IMAGE_NAMESPACE` | `growthpath` | Image namespace |
| `IMAGE_PREFIX` | `growthpath` | Image prefix |
| `TAG` | `latest` | Deploy tag |
| `PUSH` | `false` | Push images during build |
| `PUSH_LATEST` | `true` | Also tag latest |
| `PLATFORMS` | multi-arch | buildx platforms |

## URI rewrite

`scripts/rewrite-prod-uris.mjs` converts local hostnames in `.env` to Docker Compose service names and writes `.env.prod.override`.
